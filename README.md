# 🎤 Kunal Kamra App - UI Clone

This is a simple UI clone inspired by **Kunal Kamra**'s app. 
It replicates a real app's front-end design and feel, built using modern web technologies.

## 🚀 Tech Stack

- React.js
- Tailwind CSS
- Framer Motion (for animations)
